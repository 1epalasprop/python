# Python Notebooks
Ασκήσεις στη γλώσσα προγραμματισμού Python για τη Β Τάξη ΕΠΑ.Λ.

# Εκκίνηση στον Browser
Μπορείτε να χρησιμοποιήσετε αυτά τα σημειωματάρια σε διαδραστική μορφή στον
browser χωρίς να χρειαστεί να εγκαταστήσετε κάτι πατώντας την εικόνα που 
ακολουθεί: [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/1epalasprop/python-notebooks/main)

**Προσοχή:** Αν και θα μπορείτε να τρέξετε και να τροποποιήσετε τα σημειωματάρια, ό,τι αλλαγές
κάνετε μετά θα χαθούν. Την επόμενη φορά που θα το ξεκινήσετε από την παραπάνω εικόνα, θα δείτε
μόνο ό,τι υπάρχει σ' αυτό το repository.
